## 🔥 FASE 2: QUICK WINS DE TRÁFICO (Semanas 3-4)
**Objetivo:** Triplicar tráfico orgánico (40 → 120 clics/mes)

### Acciones rápidas:
1. **Optimización de títulos SEO**
   - [ ] Auditar 206 títulos actuales
   - [ ] Reescribir con formato: `{Tipo} en {Barrio} — {Precio} | {Diferenciador}`
   - [ ] Ejemplo: `Piso 3 hab en Eixample — 1.200€/mes | Calculadora gastos incluida`
   - [ ] **Métrica:** CTR de 13.5% → 18%

2. **Meta descriptions magnéticas**
   - [ ] Generar meta descriptions únicas para cada listing
   - [ ] Incluir: precio, barrio, USP (calculadoras, sin comisión)
   - [ ] Máximo 155 caracteres
   - [ ] **Métrica:** +50 impresiones/día

3. **Schema markup estructurado**
   - [ ] Implementar `RealEstateListing` schema en cada piso
   - [ ] Incluir: precio, imágenes, ubicación, características
   - [ ] Validar con Google Rich Results Test
   - [ ] **Métrica:** Rich snippets en 70% de resultados

4. **Imágenes optimizadas SEO**
   - [ ] Renombrar imágenes: `piso-3-habitaciones-eixample-barcelona-1.jpg`
   - [ ] Agregar alt text descriptivo a todas las fotos
   - [ ] Comprimir sin pérdida de calidad (WebP)
   - [ ] **Métrica:** Aparecer en Google Images

5. **Velocidad de carga**
   - [ ] Lazy loading de imágenes
   - [ ] Comprimir scripts de calculadoras
   - [ ] CDN para assets estáticos
   - [ ] **Métrica:** Core Web Vitals > 90/100

**KPI Fase 2:** 120 clics/mes (+200%)
